# Pong-Game
Pong Game - ping pong dla dwóch graczy
![ponggame](https://github.com/Spenuri77/Pong-Game/assets/174376308/6183af8d-0287-4bde-a03f-32be572cf663)

Sterowanie:
Gracz po lewej stronie:
- ruch do góry - W
- ruch do dołu - S
- superstrzał - E

Gracz po prawej stronie:
- ruch do góry - O
- ruch do dołu - L
- superstrzał - P
